# Aryan Portfolio Website

## Description

A modern, interactive portfolio website showcasing professional work and skills. The website features a stunning 3D animated avatar, interactive background effects, and a fully functional contact form. Built with cutting-edge web technologies to deliver a smooth, engaging user experience.

## Key Features

- **3D Animated Avatar**: Interactive 3D model with orbit controls, allowing users to rotate and zoom
- **Custom Animated Cursor**: Unique GIF-based cursor that follows mouse movement
- **Interactive Dot Grid Background**: Dynamic particle system that responds to mouse movement and clicks
- **Dark/Light Mode Toggle**: Seamless theme switching with smooth transitions
- **Contact Form**: Fully functional form with EmailJS integration for direct email notifications
- **Responsive Design**: Optimized for all screen sizes and devices
- **Smooth Animations**: GSAP-powered animations for enhanced user experience
- **Scroll Snap Sections**: Modern single-page application with smooth scrolling

## Technologies & Skills Used

### Frontend Framework & Build Tools
