# Aryan Portfolio Website https://portfolioaryan-chi.vercel.app/

A modern, interactive portfolio website featuring a 3D animated avatar, interactive background effects, custom cursor, and a fully functional contact form.

## Description

This portfolio website showcases professional work with cutting-edge web technologies. It features an interactive 3D model that users can rotate and zoom, a dynamic dot grid background that responds to mouse movement, a custom animated cursor, and seamless dark/light mode switching. The contact form is integrated with EmailJS for direct email notifications.

## Technologies & Skills

### Core Technologies
- **React 19** - Modern React with hooks and component architecture
- **Vite** - Fast build tool and development server
- **Three.js** - 3D graphics and WebGL rendering
- **React Three Fiber** - React renderer for Three.js
- **React Three Drei** - Helpers for 3D development

### Additional Libraries
- **GSAP** - Professional animation library
- **EmailJS** - Email service integration
- **Tailwind CSS** - Utility-first CSS framework
- **Canvas API** - Custom particle system

### Key Features
- 3D animated avatar with orbit controls
- Interactive dot grid background
- Custom animated cursor (GIF-based)
- Dark/Light mode toggle
- Responsive design
- Contact form with email integration
- Smooth animations and transitions
